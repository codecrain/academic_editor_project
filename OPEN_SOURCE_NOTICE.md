# Open Source Notice

This repository publishes the document editor runtime layer and the public
source patches used to build it. The repository is licensed under the Mozilla
Public License 2.0.

The DOCX editor source is vendored under `editor_docx/`. The HWP/HWPX editor
source is vendored separately under `editor_hwpx/`.

The private SaaS application that connects to this runtime over WOPI is a
separate larger work. Its authentication, storage, database, billing, product UI,
deployment secrets, and proprietary service code are not part of this public
runtime repository unless those files are intentionally copied here.

## Source Availability

For each production runtime release, keep these items available from this public
repository or the associated GitHub release:

- The commit SHA of this repository used for the build.
- The upstream editor source repository and ref used for the build.
- The engine asset archive URL, or the value `source` when the engine is built
  from source.
- The runtime release artifact metadata generated by `npm run package:native`.
- The source-offer note generated by `npm run source-offer`.

The service-level legal or open-source page should link to this repository and
include the exact runtime commit/ref evidence for the deployed release.

## Notices and Branding

Normal product workflows should use Tlooto or generic editor wording, not
upstream product marks. Required copyright, license, and attribution notices must
remain available in this repository and in the service-level open-source notice
page.

Do not use the public `collabora/code` development image for this SaaS runtime.
Use the source-built native runtime release path described in `README.md` and
`COMPLIANCE.md`.
